    <h2>Steps to run app</h2>

    1. run "composer install"

    2. run "npm install"

    3. run "php artisan migrate --seed"
    
    4. run "php artisan storage:link"

    5. go to storage/app/public and find .zip, extract to get default asset

    6. run "npm run prod"

    7. run "php artisan serve"
